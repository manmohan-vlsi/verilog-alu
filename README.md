# 4-bit ALU (Verilog)
This project implements a 4-bit Arithmetic Logic Unit using Verilog HDL.
## Features
- Performs addition, subtraction, AND, OR operations
- Uses 2-bit select line for operation control
## Operations
| Select | Operation |
|--------|----------|
| 00     | A + B    |
| 01     | A - B    |
| 10     | A & B    |
| 11     | A | B    |
